# Auction Smart Contract

## Overview
A decentralized auction system built in Solidity where users place bids using Ether and the highest bidder wins.

---

## Features
- Bidding system  
- Refund mechanism  
- Auction state management  
- Secure ETH transfer using call  

---

## Usage
- Deploy contract  
- Place bids using `bid()`  
- End auction and withdraw  

---

## Concepts Used
- mapping  
- modifiers  
- payable  
- require  
- call
